FAQ 14건

Q1. 자본금 5,000만이 적정한가?
PM320 유료 신고 + JEPQ 5,000만 운용 + 부동산 경매 보증금 활용 모두 가능. 단 #4 투자자문업 인가 시 1억 증자 필요.
Q2. 송도 vs 용인 어디가 유리한가?
세금 동일 (과밀억제권역 외). 임대료·접근성·이미지만 검토. 송도는 IT 이미지·인천공항, 용인은 강남 1시간·임대료 저렴.
Q3. 정관 인증료를 내야 하나?
아니오. 상법 §292 — 자본금 10억 미만 면제. 5,000만 = 면제 명확.
Q4. PM320 유료 운영하면 형사처벌인가?
양방향 채널(오픈채팅·텔레그램) 운영 시 1억 과태료 + 등록 취소 (2024-08 개정). 단방향 정보 제공(블로그·뉴스레터·웹사이트)은 신고 후 가능.
Q5. JEPQ 매수 전 무엇을 해야 하나?
W-8BEN-E 작성. 미제출 시 30% 원천징수. 한국 증권사 외화 데스크 위탁.
Q6. 외주 개발자를 사무실에 상주시켜도 되나?
위장도급 위험 (대법원 2010다106436). 발주사 지시 받으면 직접 고용 의무. 별도 공간 + deliverable 단위 + 노무사 자문.
Q7. 1인 법인에 감사가 필요한가?
자본금 10억 미만 = 의무 없음. 단 정관 명시 시 안정성 ↑.
Q8. 결산일 변경 가능한가?
가능. 단 정관 변경 + 등기(30~50만). 12월 31일 권고.
Q9. 법인세는 언제 내나?
결산 후 3개월 이내 (12-31 결산 시 3월말). 부가세는 분기 1·4·7·10월.
Q10. 가수금 무이자 대여 한도는?
31억원 (회사 내규). 약정서 작성 + 월별 잔액 관리 + 세무사 보고.
Q11. 배당은 언제 어떻게 하나?
주주총회 결의 → 지분율 분배. 자녀 배당 한도 연 2,000만원 (세제 혜택). 1인 주주는 100% 본인.
Q12. 도메인 6개 법인 양도 비용은?
가비아 명의 변경 (등록자 변경) — 무료. 단 양도세 확인 필수 (취득가 vs 양도가 차익).
Q13. 비상주 사무실에서 회의는 어떻게?
사무실별 회의실 포함/별도 다름. §6 표 참조. 직접 방문 시 확인.
Q14. 자본시장법 시행령은 어떻게 모니터링하나?
분기 1회 law.go.kr "자본시장법 시행령" 최신 시행일 확인 + 금융위 입법예고(fsc.go.kr/po040301) 향후 6개월 개정안 확인. §16 체크리스트 참조.
