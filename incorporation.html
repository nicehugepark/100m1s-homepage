  • 외부 투자 유치 옵션 보전. VC는 사실상 주식회사만 투자.
  • 유사투자자문업·금융 관련 신고 표준. 자본시장법 §101 신고 시 주식회사가 표준.

  • 법인카드·법인 계좌 발급 표준. 은행 실무에서 가장 막힘 없음.

1.2 주식회사의 단점·반대 의견

  • 외부 감사 의무는 자본금 10억 미만이면 면제 (1,000만 원 자본금 시 무관).
  • 결산공고 의무: 매년 결산 후 인터넷 홈페이지 또는 일간신문 공고 — 운영 부담 존재. 정관에 "회사 인터넷 홈페이지에 공고"로 정해 비용·실무 최소화 가능.
  • 1인 창업자 + 다목적 운영 시 주주총회·이사회 의무 (서면결의 가능하나 회의록 필수) 행정 부담 → 세무사·법무사 매월 기장료에 포함되도록 협상 권고.
SECTION 02

사전 결정 사항

법무사·세무사 의뢰 전에 창업자가 직접 결정해야 할 항목입니다.

2.1 회사명 (상호)

  • 한국 상법·등기 실무: 상호는 한글·한자·아라비아 숫자·영문 사용 가능 (대법원 등기 예규).
  • 등기소(인터넷등기소 iros.go.kr) 사전 상호조회 필수. 동일·유사 상호 충돌 검토.

2.2 본점 주소

비상주 사무실 사용 시 본점 등기 가능 여부를 사업자에게 사전 확인. §6 송도 vs 용인 비교 참조.

2.3 자본금

  • 권고: 1,000만 원. 상법 개정으로 100원 단위까지 가능하나, 은행·거래처 신뢰도·VC 실사 시 1,000만 원 이상이 표준.
  • 등록면허세 = 자본금 × 0.4% (비과밀억제권역) 또는 ×1.2% (과밀억제권역).
  • 너무 낮으면(예: 100만 원) 거래처 신뢰도 하락 + 향후 증자 시 등록면허세 추가. 너무 높으면(예: 1억 원) 등록면허세만 80만~120만 원.

2.4 주주 구성

  • 옵션 A — 단독 100% (1인): 단순, 의사결정 빠름, 향후 자녀·배우자에게 증여로 분산 가능.
  • 옵션 B — 가족 분산: 자녀 증여 공제 활용·배당 분산·상속 절세 효과. 단, 미성년 주주의 법정대리인 의결권 행사 조항 등 정관 복잡도 증가.
검증 권고 옵션 B(가족 분산) 선택 시 증여세·자금출처·증여의제이익 검토를 위해 세무사·법무사 자문 필수.

2.5 임원 구성

자본금 10억 미만 → 이사 1인 + 감사 미설치 가능 (상법 §383). 권고: 대표이사 1인. 향후 필요 시 이사 추가 등기.

2.6 사업 목적

정관에 5~10년 후 가능 사업까지 포괄적으로 기재. "위 각 호의 부대 사업" 조항 필수. 신규 사업 추가 시 정관 변경 등기 비용(등록면허세 + 법무사 대행료)이 발생합니다.

2.7 발행주식·액면가

  • 발행할 주식의 총수: 100,000주 (미래 증자 여유분).
  • 설립 시 발행 주식 수: 10,000주.
  • 1주 금액(액면가): 1,000원 (1,000원 × 10,000주 = 1,000만 원).
  • 주식 종류: 보통주만. 우선주·종류주는 외부 투자 유치 시 도입.

2.8 사업연도(결산일)

1월 1일 ~ 12월 31일이 한국 표준. 해외 자회사·특수 업종이 아니면 이 외 선택 비권장.

자주 하는 실수 + 주의사항

9.1 자본금

  • 너무 낮게 (10만 원) → 거래처·은행 신뢰도 하락. 향후 증자 시 등록면허세 추가.
  • 너무 높게 (1억 원) → 등록면허세 80만~120만 원. 회수도 어려움.
  • 권고: 1,000만 원.

9.2 사업 목적

  • 현재 사업만 기재 → 향후 신규 사업 추가 시 정관 변경 등기 비용 발생.
  • 5~10년 후 가능 사업까지 포괄적으로 기재. "위 각 호의 부대 사업" 조항 필수.
  • 유사투자자문업 누락 → 자본시장법 위반 (형사처벌).

9.3 1인 법인 임원·감사

  • 자본금 5억 미만임에도 감사 선임 → 불필요한 비용·행정 부담.
  • 자본금 10억 미만 → 이사 1인 + 감사 미설치 (상법 §383).

9.4 도메인·계정 명의이전

  • 법인 설립 후에도 개인 명의 유지 → 법인 자산 누락, 향후 양도소득세 발생 가능성.
  • 등기 직후 즉시 명의이전. 가비아·GitHub·Vercel·Google Workspace.
  • 명의이전 시 SaaS 데이터 마이그레이션 손실 위험 → 사전 백업.

9.5 법인 통장 개설 거절

  • 금융사고 우려로 신규 법인 통장 개설 거절 사례 증가 (2023~2026 강화).
  • 사전 지점장 면담 + 사무실 방문 사진 + 임대차계약서 + 사업계획서 지참.
  • 주거래 은행 1곳 + 보조 2곳 동시 진행 (1곳 거절 대비).

9.6 4대 보험 시점

  • 사업자등록 직후 즉시 가입 → 매출 0인 시점부터 보험료 부담 (월 70만~100만 원).
  • 첫 매출 발생 또는 첫 급여 지급 직전 가입 (단, 등기 후 14일 이내 사업장 신고는 의무).

9.7 비상주 사무실 본점 이전

  • 본점 이전 시 자본금 × 0.4% 등록면허세 추가. 비상주 사무실 폐업·이사 시 동일.
  • 장기 계약 + 사업자 신뢰도 사전 확인.

9.8 가족 분산 주식 (옵션 B)

  • 자녀 자금출처 증빙 누락 → 차명 의심 + 증여의제이익 추징.
  • 자녀 명의 계좌 → 부모 계좌 → 발기인 대표 계좌 흐름 + 증여세 신고서 보관 5년.
  • 미성년 주식 인수 → 법정대리인(부모 양친) 동의서 필수. 정관에 의결권 대리행사 조항.

9.9 정관 무료 템플릿

  • 인터넷 무료 정관 사용 → 1인 창업·가족법인·유사투자자문업 등 특수 조항 누락.
  • 법무사·세무사 의뢰 (10만~30만 원). 향후 세무조사·분쟁 시 핵심 증거.

9.10 첫 결산·법인세 신고

  • 결산일(12/31) ~ 법인세 신고(다음 해 3/31) 미준비 → 가산세 (무신고 20% + 납부불성실 8.03%/년).
  • 세무사 기장 계약 + 분기 1회 미팅. 12월 중순 결산 사전 점검.

9.11 법인 인감 vs 대표이사 인감

혼용 금지. 법인 인감 = 등기·계약·세금. 사용인감 = 일반 업무. 대표이사 인감 = 개인 (법인 등기 신청 시 인감증명서 첨부).

FAQ

Q1. 자본금 1,000만 원이 너무 적은가?
거래처 실사·VC 시드 라운드 기준으로는 충분합니다. 다만 첫 1년 운영비(월 100만 원 × 12 = 1,200만 원)를 초과하므로, 가수금(대표 → 법인 무이자 대여, 한도 31억 원)으로 운영자금 보충이 표준입니다. 자본금을 1억 원으로 올리면 등록면허세 80만 원 추가 + 회수 어려움.
Q2. 1인 단독 vs 가족 공동창업, 어느 쪽이 유리한가?
세무 절세는 가족 공동이 유리(배당 분산, 자녀 증여 공제). 그러나 의사결정 속도·정관 단순성은 1인 단독이 유리. 미성년 자녀 주주 시 정관에 의결권 대리행사 조항 + 모든 주주총회 의사록에 법정대리인 서명 필수 → 행정 부담 증가. 권고: 첫 1년 1인 단독 → 운영 안정 후 자녀·배우자에게 증여. 단, 상속 절세는 시간이 오래 걸리므로 자녀 어린 단계에 분산하는 것이 통상 유리. 세무사 검증 권고.
Q3. 비상주 사무실에서 본점 이전 시 비용은?
본점 이전 등기 = 자본금 × 0.4% (비과밀) 또는 1.2% (과밀) 등록면허세 + 지방교육세 20% + 법무사 대행료 10만~20만 원. 자본금 1,000만 원 기준 약 5만~30만 원. 추가로 사업자등록 정정·임대차계약서·통장 주소 변경 등 행정 작업 약 1주.
Q4. 첫 결산은 언제, 무엇을 준비하나?
결산일 12/31 → 법인세 신고 다음 해 3/31까지. 부가세 신고는 1/25, 7/25 (반기). 준비물: 매출·매입 세금계산서, 법인카드 사용내역, 통장 거래내역, 급여대장, 4대보험료, 가수금 약정서·상환내역, 재고. 세무사 기장 계약 시 대부분 자동.
Q5. 법무사·세무사 위임 vs 직접 진행?
정관 작성 + 등기 신청은 법무사 위임 강력 권고 (20만~50만 원). 직접 진행 시 보정 명령 → 처리 지연 + 재신청 비용. 사업자등록·세무 기장은 세무사 위임 필수 (월 15만~30만 원). 직접 시 부가세·법인세 가산세 위험. 노무사 위임은 4대보험 신고 1회성 또는 분기 점검 정도면 충분.
Q6. 여러 사업을 한 법인에서 운영해도 되나?
가능하나 분리 검토 필요. 한 법인 다목적 운영 시 (1) 사업 부문별 회계 분리 어려움, (2) 유사투자자문업 등 규제 영역의 규제가 다른 사업까지 확장 검토 필요, (3) 한 서비스의 법적 분쟁이 다른 서비스 자산에 영향. 권고: 첫 1년은 한 법인에서 모두 운영하며 매출 발생 사업 식별 → 매출 1억 이상 사업은 별도 법인 분리 검토. 법무팀 검증 권고.
Q7. 증자(자본금 증액)는 언제 어떻게?
외부 투자 유치·신규 사업 자금 필요 시. 절차: 이사회 결의(1인 이사면 결정서) → 신주 인수 청약·납입 → 증자 등기. 비용: 증자액 × 0.4~1.2% 등록면허세 + 법무사 대행료. 자본금 1,000만 → 1억 증자 시 약 50만~150만 원.
Q8. 투자 받기 좋은 법인 형태·구조는?
주식회사 + 보통주 단일 종류 + 정관에 신주 발행 한도 충분 + 주식양도제한 조항 합리적 구조. VC는 우선주 발행을 통해 투자하므로 정관에 우선주 발행 가능 조항 사전 포함 권고. 자녀 70% 등 가족 지분이 큰 구조는 VC 실사 시 거버넌스 우려로 디스카운트 또는 거절 사유. 외부 투자 가능성이 있으면 옵션 A (대표 100%) → 추후 분산.
Q9. 도메인 개인 → 법인 명의이전 비용·시간?
가비아 기준 명의이전 신청서 + 법인 사업자등록증 + 법인 인감증명서 + 위임장 제출. 수수료 0~5,000원/건. 처리 기간 1~3영업일. 주의: 일부 도메인은 명의이전 시 WHOIS 정보 변경에 따라 1년간 도메인 이전 불가 락(transfer lock) 적용. 등록 후 60일 내 이전은 가비아도 제한 가능.
Q10. 법인 인감 vs 대표이사 인감, 어떻게 구별·사용?
  • 법인 인감 (등기소 등록): 회사 명의 등기·중요 계약·세금 신고. 등기소 인감증명서 발급. 분실 시 변경 절차.
  • 법인 사용인감: 일반 업무 (영수증·비공식 문서). 등기소 등록 X.
  • 대표이사 (개인) 인감: 개인 신분 증명. 법인 등기 신청 시 인감증명서 첨부. 주민센터 발급.
권고: 법인 인감은 금고 보관, 사용인감은 일상 업무, 대표이사 인감은 별도 보관. 3인감 절대 혼용 금지.
