§11. 개미털기 (Shake-out) 심층 분석

개미털기
분배 도중 가장 빈번하고 가장 치명적인 변칙. 의도적 폭락으로 개미를 손절시키고 저점에서 재매수, 재상승으로 차익을 극대화한다. 토구사 자기 학습 매뉴얼 핵심 챕터.

11.1 정의

작전 세력이 분배 도중 의도적 단기 폭락(-3~-7%)을 유도하여 개미를 손절시키고, 저점에서 재매수한 뒤 재상승으로 잔여 분배 + 추가 차익을 극대화하는 변칙.

개미털기
핵심: 작전 세력의 실제 비중은 변동 없음. 매도 호가 던지기 + 즉시 재매수 = 일시적 호가창 무너뜨림. 개미만 손절선 트리거.

2.10 인물 심리 분석 (Psychology Profile)

각 인물이 사이클 단계별로 갖는 인지·감정·의사결정 패턴. PM320 사용자가 "왜 이 함정에 빠지는가"를 메타인지하기 위한 자료.

방장 (풍문책)

  • 자기 인지: "나는 시장 분석가, 단지 정보 전달자". 자신의 행위가 시세조종이라는 자각 약함 (또는 의도적 부인).
  • 수익 구조: 회원 결제금 (월 50~300만원 × 5천명 = 월 25~150억원) + 자금책 분배 수익 (10~30%).
  • 위험 인지: 검찰·금감원 추적 위험은 낮다고 판단 — 다수 자연인 분산·해외 서버·VPN 사용. 실제 기소 사례 적음.
  • 반복 동기: 한 종목 사이클 (D-21~D+30) ≈ 2개월. 연 6회 사이클. 종목당 순익 5~20억 추정.

자금책

  • 자기 인지: "투자자". 작전팀·방장과의 관계는 "비즈니스 파트너"로 정의.
  • 수익 구조: 매집-띄움 차익 (자기자본 30~100억 × 사이클당 +30~50% = 사이클당 10~50억).
  • 위험 인지: 자기 명의 계좌 사용 안 함 (차명 계좌, 가족·지인 명의). 직접 매매 행위 부인 가능 구조.

작전팀 (실시간 컨트롤러)

  • 자기 인지: "전문 트레이더". 호가 컨트롤·분할 매도 기술 보유.
  • 수익 구조: 자금책 고용 (월 고정 + 사이클 성과). 다수 계정·휴대폰 운용.
  • 위험 인지: 본인은 매수·매도만 실행, 풍문 유포는 방장이 함 → 책임 분산.

유료 회원 (1차 피해자)

  • 진입 심리: "이번에는 다르다", "방장 적중률 87%", "유료 회원만의 정보". 결제금 자체가 sunk cost로 작동 → 이탈 어려움.
  • 매수 직후 심리: 가격 +10% 시 "역시 방장은 다르다" 확신 강화. 다른 회원 인증 게시에 동조감.
  • 고점 심리: 방장이 "추가 매수" 권고하면 평균단가 낮추기 위해 추가 매수. 이 시점이 작전팀 분배의 핵심 출구.
  • 하락 심리: 방장이 "홀딩·물타기" 권고하면 손절 못함. 결제금 = 신뢰 lock-in 메커니즘. 환불 시점에는 이미 -25% 손실.
  • 재가입 심리: 새 방 개설 시 "이번 방장은 진짜일 것" 또는 "지난번 손실을 만회하자" → 도박 회복 심리. 학습 안 됨.

일반 개미 (2차 피해자)

  • 진입 심리: 카페·디시·X에서 캡처 보고 "지금 들어가야" → FOMO (Fear of Missing Out). 호가창 폭주가 추격매수 강화.
  • 매수 직후 심리: 호가창 매수 1·2호가 폭주 → "강한 매수세" 인식. 그러나 이는 작전팀 분할 매도의 흡수 호가일 가능성.
  • 고점 심리: 캡처에 "VIP방 추천" 표시 → 권위 위장 효과. 본인이 "정보 우위에 있다"는 착각.
  • 하락 심리: D+1 갭 상승 후 급락 시 "조정"으로 인식. 손절 늦음. 본인이 작전팀의 마지막 출구라는 인식 부재.

무관 종목투자자 (3차 영향자)

  • 진입 안 함: 본 종목 보유 안 함. 다만 본인 보유 종목이 갑자기 작전 타깃이 되면 단기 변동성 노출.
  • PM320 관점: 본 시점에 매수 진입하면 자동으로 1차/2차 피해자 군에 편입. PM320 페널티가 진입 차단의 마지막 방어선.

3. 다른 케이스의 디지털 채널 보강 (토구사 메인 통합용)

변칙
토구사 메인 케이스 작성 후 본 섹션을 cross-reference하여 흡수. 각 케이스 끝 변칙 라인은 §2.6.1 박스의 케이스별 적용판.

3.1 케이스 1 (호가 작전형) — 디지털 채널 부분

  • 호가 작전과 동시에 카페·텔레그램에 "지금 들어가야" 게시 (호가 폭주 5~10분 전 사전 시그널)
  • 식별: 게시 시각 vs 호가 폭주 시각 lag 분석. 일관되게 게시가 선행 → 사전 정보 의심.
  • 변칙: (a) 호가 충격 직전 채널 침묵 1~2분 (개미털기 SHK-1과 동일 메커니즘). (b) 호가 회복 직후 "체결 강도 강함" 자랑 게시 → 추격매수 압력 강화.

3.2 케이스 2 (단기 모멘텀 추적형) — 디지털 채널 부분

  • 일반 개미가 자발적으로 게시 (작전 의도 없음). 단, 작전팀이 이 자발적 분위기에 편승하여 띄움.
  • 식별: 카페 게시 작성자 다양성 높음 (unique > 70%) + 시점 분산 → 자발적. 그러나 호가는 조작 가능.
  • 변칙: (a) 자발적 분위기 형성 후 작전팀 인플루언서 1~2명 cross-promotion 진입 → 가속도 인공 부풀림. (b) 카페 작성자 다양성 높지만 댓글 봇 비율 ↑ (자발 게시 + 인공 댓글 hybrid).

3.3 케이스 4 (허위공시형) — 디지털 채널 부분

  • 공시 직후 ±5분 내 카페·텔레그램·X 동시 다발 게시. 공시-게시 lag이 비정상적으로 짧으면 사전 입수 의심.
  • 식별: 공시 timestamp vs 첫 게시 timestamp 차이가 < 60초 → 봇 또는 사전 정보 입수.
  • 변칙: (a) 공시 D-1 시점 "내일 큰 거 옵니다" 사전 귀띔 게시 → 공시 후 자기 게시 cherry-pick으로 권위 강화. (b) 공시 정정·취소 후에도 "재공시 확실" 풍문 유지 → 분배 추가 사이클 확보.

3.4 케이스 5 (펌프앤덤프 외국인) — 디지털 채널 부분

  • 외국계 증권사 매수 보고서 + 카페·블로그 동시 번역·인용. 영어 원문 출처 불분명한 경우 다수.
  • 식별: 원문 영어 자료의 실재 여부 확인 (이시카와가 직접 검증 불가 시 "데이터 부족" 명시).
  • 변칙: (a) 가짜 외국계 보고서 PDF 합성 (헤더만 진짜, 본문 조작). (b) 영어 원문 링크 dead URL 또는 paywall로 검증 차단.

3.5 케이스 6 (테마주 회전형) — 디지털 채널 부분

  • 테마 키워드 (예: "AI 반도체", "전쟁재건") 카페·X 동조 급증. 인플루언서 endorsement 다발.
  • 식별: 테마 키워드 가속도 + 작성자 다양성 + 인플루언서 계정 신뢰도 (과거 픽 적중률).
  • 변칙: (a) 테마 내 종목 rotation — 1번 종목 분배 끝나면 같은 테마 2번 종목으로 시그널 이동. (b) 테마 키워드 자체를 작전팀이 만들어 SNS에 사전 살포 (테마 manufactured).

3.6 케이스 7 (CFD 청산 트리거형) — 디지털 채널 부분

  • CFD 다종목 동시 청산 직전, 풍문방에서 같은 시각 동일 종목군 시그널 발사 (청산 충격 흡수 시도).
  • 식별: 동일 시각 N개 종목 동시 게시 + CFD 보고 데이터 cross-reference.
  • 변칙: (a) 청산 충격 직전 "급락은 매수 기회" 사전 게시 → 청산 폭락을 "지지선 확인"으로 프레이밍. (b) 청산 후 "이제 바닥" 풍문으로 재진입 유도 → 작전팀 추가 분배.

3.7 케이스 9 (바이오 임상 결과 사전 유출) — 디지털 채널 부분

  • 임상 결과 공시 D-3~D-1 시점 텔레그램 VIP방에서 사전 픽. 공시 후 가격 +30% 갭 상승 → 사전 정보 입수 강한 의심.
  • 식별: 공시 timestamp vs VIP방 픽 timestamp. 사전이면 미공개 중요 정보 이용 (자본시장법 제174조 추정 — 법무 검증).
  • 변칙: (a) 임상 실패 사전 입수 시 공매도 또는 풋옵션 시그널 발사 (역방향 작전). (b) 임상 결과 모호 시 "긍정 해석" 풍문 우선 유포 → 단기 띄움 후 분배.

3.8 케이스 10 (퇴출 직전 작전) — 디지털 채널 부분

  • 상장폐지 사유 발생 직전 "회생 시그널" 풍문 (M&A·자본 유치설). 마지막 분배.
  • 식별: 회사 공시·실적·재무 vs 풍문 내용 일치도. 풍문이 회사 펀더멘털과 무관하면 작전 가능성 매우 높음.
  • 변칙: (a) 정리매매 기간에도 "거래재개 확실" 풍문 유지 → 마지막 매수자 확보. (b) 동일 풍문책이 퇴출 종목 cycle 반복 — 새 종목으로 같은 패턴 재가동.

4. PM320 스코어 페널티 후보 — 정량 작전 시그널 11항 (v1.2 격상)

방어
본 정량 시그널은 5요소 도메인 강점 활용. 토구사·휴지 4자 의논 시 PM320 스코어 페널티로 합류 권고. v1.2: §4.1~§4.6 (풍문 캠페인 6항) + §4.7~§4.11 (개미털기·분배 5항 신규) + §4.12 종합 캡.
데이터
임계 가설은 백테스트 v3에서 30일 후 재조정 권고. 매직 넘버 (N=10, M=30%, ±5분 등)는 모두 가설 단계.

4.1 단일 출처 다발 (Single-Source Burst)

  • 정의: 1개 채널에서 N+ 게시 (24h 내), unique author < M
  • 임계 가설: N=10건 + unique < 30% → 페널티 -15
  • 근거: 자작극 의심. 풍문 캠페인 1차 신호.

4.2 시점 동조성 (Time Synchronization)

  • 정의: 여러 채널이 ±5분 내 동일 종목 게시
  • 임계 가설: ≥3개 채널 동시 → 페널티 -20
  • 근거: 자연 발생 확률 매우 낮음. 캠페인 강한 시그널.

4.3 작성자 다양성 부족 (Author Diversity)

  • 정의: 24h 게시 N건 중 unique author 비율
  • 임계 가설: unique 비율 < 30% → 페널티 -10
  • 근거: 자발적 분위기 vs 조작 분위기 구분 핵심 지표.

4.4 가속도 vs 절대량 (Acceleration over Absolute Volume)

  • 정의: 게시·댓글 가속도 양수 + 절대량 폭증
  • 임계 가설: 가속도 > 평소 5배 + 절대량 > 평소 10배 → 페널티 -15
  • 근거: 절대량만 보면 작전 dump 함정. 가속도 함께 봐야 신선 모멘텀과 작전 분배 구분.

4.5 댓글·반응 진정성 (Comment Authenticity)

  • 정의: 봇 패턴 — 반복 문구, 동일 시각 다발, 무내용 답글
  • 임계 가설: 봇 패턴 비율 > 40% → 페널티 -15
  • 근거: 인공 분위기 조성. 일반 개미 추격매수 유도.

4.6 Inflow Funnel (무료 → 유료 → 결제 종목)

  • 정의: 무료 채널에서 유료 채널 결제 광고 → 결제 종목 시그널 흐름 식별
  • 임계 가설: funnel 식별 시 페널티 -20
  • 근거: 작전 비즈니스 모델 핵심. 유료 회원 자체가 분배 대상.

4.7 권유 vs 행위 괴리 (Advise-Action Discrepancy) — 신규 v1.2

  • 정의: 방장 게시 텍스트("매수"/"홀딩" 키워드) vs 종목 호가창 분배 패턴 (잘게 쪼갠 매도호가 누적)이 정반대
  • 임계 가설: 권유 게시 ≥3회 + 매도호가 잘게쪼갬 비율 ≥60% (당일 30분 단위) → 페널티 -15
  • 근거: §2.6.1 C1 변칙 정량 외화. 작전팀이 디지털에 남기는 가장 강력한 자기 모순 흔적. 사후 부인 어려운 증거
  • 출처: §2.12 W1 / 케이스 3 + 케이스 1·2·4·6·8·10 분배 단계 공통

4.8 방장 침묵 직전 폭락 (Pre-Crash Silence) — 신규 v1.2

  • 정의: 방장 게시 빈도가 평소 baseline 대비 -80% 떨어진 직후 5분 내 가격 -3%↓
  • 임계 가설: 빈도 -80% (3~5분 윈도우) + 직후 5분 가격 -3% → 페널티 -10
  • 근거: §2.6.1 C3 변칙 정량 외화. "시그널 부재 자체가 신호" — 평상시 활발 게시하던 채널이 폭락 직전 갑자기 침묵하면 작전팀이 분배 마무리 단계
  • 출처: §2.12 W2 / 케이스 3 + 분배 단계 일반

4.9 저점 재매수 유도 (Bottom-Fish Bait) — 신규 v1.2

  • 정의: "지지선·기회·추가 매수" 키워드 게시 시점이 가격 -3% 시점 ±2분 내, 직후 5분 가격 미반등
  • 임계 가설: 권유 게시 시점 ±2분 + 5분 후 가격 미반등 → 페널티 -15
  • 근거: §2.6.1 C4 변칙 정량 외화. 진짜 매수 권고면 본인이 매수 → 가격 일시 반등이 자연스러움. 권유 후에도 가격 무반응 = 본인 매수 안 함 = 분배 함정
  • 출처: §2.12 W3 / 케이스 3 + 분배 단계

4.10 환불·항의 게시 삭제율 (Censorship Rate) — 신규 v1.2

  • 정의: 카페·텔레그램에서 "환불·손실·사기" 키워드 게시의 60분 내 삭제율
  • 임계 가설: 삭제율 ≥70% → 페널티 -20
  • 근거: §2.6.1 C6 변칙 정량 외화. 정상 운영 채널은 항의 게시도 보존. 70%↑ 삭제 = 작전 사이클 은폐. 가장 강한 신호 중 하나
  • 운영 주의: 카페 크롤러 1시간 폴링 + 게시 archive 보존 필수. 삭제된 게시는 evidence asset
  • 출처: §2.12 W4 / 케이스 3 + 케이스 8 사모펀드 환매 중단 단계

4.11 종목 rotation 광고 lag (Rotation Ad Lag) — 신규 v1.2

  • 정의: 기존 종목 폭락 후 신규 종목 광고가 동일 방장 채널에 등장하는 시간
  • 임계 가설: 폭락 -10% 발생 후 ≤24h 내 신규 픽 공개 → 페널티 -10
  • 근거: §2.6.1 C7·D4 변칙 정량 외화. 정상 운영자라면 사이클 종료 후 회고·반성 기간 필요. 24h 내 새 픽 광고 = "분배 출구를 새 종목으로 이전" 패턴
  • 운영 주의: 방장-종목 history DB 누적 시 가능. 사이클 N회 누적되면 confidence 강화
  • 출처: §2.12 W5 / 케이스 3 + 케이스 6·10 rotation

4.12 종합 페널티 캡 (v1.2 격상)

  • 위 11항 합산 페널티 최대 -75 (v1 -50에서 격상. 5항 신규 추가로 합산 가능 페널티 -150 → 캡 필요)
  • 페널티 -50 이상 누적 시 PM320 후보 자동 제외 (v1 -30에서 격상)
  • 페널티 -65 이상 누적 시 휴지 작전 의심 강제 알림 (모바일 푸시) + 카드 UI 경고 표시 (design-lead 협의)
  • 페널티 발생 시 휴지 알림 + 작전 의심 플래그
  • 4자 의논 통합 시 토구사 페널티 11건 + 본인 11건 = 22건 → 통합 합산 cap -150 권고 (별도 합의 의제)
